# 👁️ VLA 发帖规范 (Vision-Language-Action Rules)

**存放目录**：`posts/vla/` 或其相关的多级嵌套下级子目录（如 `posts/vla/vla-2025/`）  
**首页挂载 `index.html` 的 `data-category` 属性值**：`vla`

## 内容范畴要求
专属前沿跟踪领域：一切关于 Vision-Language-Action (视觉-语言-动作) 端到端模型的最新论文解读（如 OpenVLA, π0, GROOT, Octo 等）。关注具身大模型的多模态推理和直接输出执行能力的演进路线与模型特点。

## 版块专属标签 (Tag)
生成本类文章和对应首页卡片时，应配置以下高频研究标签：
- `论文解读`
- `前沿动态`
- `模型评测`
- `多模态`
- `VLA架构`

## 专项排版期望
1. **不可撼动的溯源原则**：由于本板块是对最前沿科学文献的分析，在文章的页眉（标题下方）或结尾**必须**附上引用的论文链接（例如 arxiv 的 HTTPS 完整链接），或者提供开源模型的 Hugging Face / GitHub 准确连接。
2. **拆解阅读模式**：为提升读者获取信息的效率，AI 撰写 VLA 模型分析时，一律采用标准的子版块独立解析模式：
   - 痛点与背景 (解决什么核心阻碍)
   - 核心方法 (Methodology，他们用了什么创新架构)
   - 实验验证 (关键指标提示了多少性能，超越了什么基线模型)
   - 局限性分析 (不可刻意隐瞒缺陷)
3. **系列归档逻辑**：如果是属于统揽大类目的巨型综述下拆分的单篇子论文（如作为 2025 综述的一部分），生成的单篇文章应当放置于 `posts/vla/vla-2025/` 子文件夹中，同时页面左上方的导航按钮需确保指向上层综述（`../vla-2025.html`）而非强行退回最外层主页。
