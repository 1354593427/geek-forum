# 🧠 算法与架构发帖规范 (Algorithm & Architecture Rules)

**存放目录**：`posts/algo/`  
**首页挂载 `index.html` 的 `data-category` 属性值**：`algo`

## 内容范畴要求
本板块深度探讨强化学习 (RL)、模仿学习 (IL)、强化学习调参策略、大模型推理底层框架、分布式训练架构（如 Megatron, DeepSpeed 等分布式切分策略）与高效通讯方案等硬核算法向的内容讨论。

## 版块专属标签 (Tag)
在新建文章的最顶部以及其对应的 `index.html` 卡片中，应选择或标记以下核心关键词：
- `核心算法`
- `分布式架构`
- `性能调优`
- `训练理论`
- `强化学习 RL`
- `GPU通讯`

## 专项排版期望
1. **数学与逻辑证明**：本板块是全论坛最深度的数学推导与理论聚集地。如果涉及复杂的数学理论或公式（如 PPO, SAC 更新规则），尽量使用清晰易读的结构化表述，建议将公式放入内联或者区块高亮段落加以标注和换行说明。
2. **架构拆解**：如果涉及网络架构（Transformer 等），文章应按照标准的科学步骤**结构化地拆解**：输入嵌入层 (Embedding) -> 特征提取层 (Attention/MLP) -> 解码生成层等模块进行分段论述。
3. **可读性**：任何引用到诸如 “Actor-Critic” 或算法专有名词的内容，请保持首字母大写等专有词汇严谨格式。
